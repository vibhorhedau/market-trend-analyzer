What it does
Features
↗

Live stock data

Fetches 6 months of OHLCV data for AAPL, GOOGL, MSFT and TSLA via yfinance API.

▲

Trend detection

MA7 vs MA30 crossover strategy automatically flags Bullish or Bearish signals per stock.

■

Real estate analysis

Rent yield, price-to-rent ratio, and market buy/rent signals for 4 Indian cities.

❖

SQL analytics

Raw SQL queries for volatility ranking, monthly averages, and best yield city detection.

▩

SQLite storage

Normalized schema with auto-incremented IDs, timestamps, and clean relational design.

◆

Visualizations

Interactive Chart.js dashboard with tabs, KPI cards, and live signal table.
